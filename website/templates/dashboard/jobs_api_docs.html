Jobs API Documentation

This page documents the scraper-backed job endpoints, the manual-script URL cache API, and the apply-service workflow that now runs inside Kumquat.

Base URLs: /api/jobs/, /api/job-postings/, /api/manual-scripts/urls.json.

1. Search Jobs

Method: GET

Endpoint: /api/jobs/

Description: Returns a paginated list of scraper-backed job postings.

Example

GET /api/jobs/?search=python&location=remote&page=1&page_size=12

2. Job Posting Dashboard API

Method: GET

Endpoint: /api/job-postings/

Description: Returns the scraper dashboard listing used by the internal job operations UI.

3. Manual Script URL Cache

Method: GET

Endpoint: /api/manual-scripts/urls.json

Description: Returns cached source URLs extracted from files under website/manual_scripts/.

Example

GET /api/manual-scripts/urls.json?script=submit/greenhouse_apply.py&page=1&page_size=25

Response Shape

{
  "count": 1,
  "page": 1,
  "page_size": 25,
  "docs_url": "https://kumquat.info/api/jobs/docs",
  "filters": {
    "script": "submit/greenhouse_apply.py"
  },
  "results": [
    {
      "script_name": "submit/greenhouse_apply.py",
      "source_name": "GREENHOUSE_URL",
      "url": "https://job-boards.greenhouse.io/example/jobs/123",
      "file_modified_at": "2026-04-21T18:00:00+00:00"
    }
  ]
}

4. Apply Service

Methods: GET, POST

Endpoint: /api/job-postings/<id>/apply-service/

Authentication: Django session login.

Description: Creates or resumes an application run, stores per-domain credentials, and returns run state for polling.

Queue A Run

POST /api/job-postings/42/apply-service/
{
  "action": "start",
  "siteUsername": "applicant@example.com",
  "sitePassword": "Secret123!"
}

Poll Run State

GET /api/job-postings/42/apply-service/?runId=91

5. Internal Operations Pages

  • /api/scrapers/ manages scraper definitions and run history.
  • /api/scrapers/runs/<id>/ shows individual scraper run detail.
  • /api/manual-scripts/ manages the manual script queue and worker controls.
  • /api/manual-scripts/urls/ shows the cached manual-script URL inventory.
  • /api/job-postings/<id>/apply/ opens the application manager UI for a job.
