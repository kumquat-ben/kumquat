# Wiki Seed

This directory contains the initial GitHub wiki pages for `kumquat-ben/kumquat`.

GitHub only exposes the `.wiki.git` repository after the wiki has been initialized with a first page in the web UI. If `https://github.com/kumquat-ben/kumquat/wiki` redirects back to the repo root, create the first blank wiki page in GitHub once, then publish this directory.

After the wiki exists, publish with:

```bash
./scripts/publish-github-wiki.sh
```
