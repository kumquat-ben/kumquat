# Kumquat Chain Modules Documentation

This directory contains documentation for the modules that make up Kumquat Chain.

## Core Modules

- [Consensus Module](consensus.md): hybrid Proof-of-Work and Proof-of-History
- [Network Module](network.md): peer-to-peer communication and synchronization
- [Storage Module](storage.md): blockchain data, state, and object persistence
- [Cryptography Module](crypto.md): cryptographic primitives
- [Mempool Module](mempool.md): prioritized pending-transaction handling

## Additional Modules

- [Smart Contract Module](smart_contract.md): contract execution support
- [Wallet Module](wallet.md): key management and signing
- [API Module](api.md): node and client interfaces
- [CLI Module](cli.md): command-line interaction

## Design Principles

- modular responsibilities
- testable subsystems
- performance-aware implementation
- security for minted and transferred units
